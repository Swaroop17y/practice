some options with ls are(ls -la, la -l, ls -f, ls -r, ls -t)

-l -list in long format
-a -all files including those which begin with dot
-f -it is useed to classify the directories,executables. it will append indicator(one of*/=>@\) to entries
-i -list the inode number in the front column
-R -recursively list all directories & subdirectories
-r -display the reverse order
-t -sorts files by access time 
-d -shows the names of directories and not their contents
-h -indicates Human readable. this mentions file size  
-S - shortfiles by size
